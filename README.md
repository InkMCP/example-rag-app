# RAG Application

Learn how to deploy a retrieval-augmented generation app with Ink MCP.

## Tutorial

Full step-by-step tutorial: [docs.ml.ink/examples/ai-and-mcp/rag-app](https://docs.ml.ink/examples/ai-and-mcp/rag-app)

### Prerequisites

- An [Ink](https://ml.ink) account
- Ink MCP connected to your AI agent ([setup guide](https://docs.ml.ink/quick-start))

### Quick Deploy

1. Connect Ink MCP to your agent
2. Prompt: "Build a RAG app that lets users upload documents and ask questions about them. Use LangChain with an in-memory vector store and Claude as the LLM. Build the frontend with React."
3. Prompt: "Deploy with Ink"

## About Ink

[Ink](https://ml.ink) is a deployment platform built for AI agents. Connect the Ink MCP server to any AI coding agent and deploy applications, manage domains, databases, and infrastructure — all through natural language prompts.
